# DevOps API

Backend API for DevOps application, which is a bootcamp based website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run Application

```
#Run in dev mode
npm run dev

#Run in prod mode
npm start
```
